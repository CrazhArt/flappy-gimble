# Flappy Gimble

## Repository
Link to respository: [Click here](https://github.com/CrazhArt/flappy-gimble/tree/main)

## Description
A simple game that utilizes simple physics and event mechanics.
Based around a vampire character for the purpose of entertainment.

## Features
- Feature 1 - Character and obstacle classes
  - Creating classes to determine what objects are the character and the obstacles, as well as their characteristics
- Feature 2 - Click to jump
  - Implement a way to detect user input and execute the mechanic to get the character to jump
- Feature 3 - Obstacles
  - Hurts the player and ends the game loop
- Feature 4 - Game loop
  - The main loop executed when the player starts the game; Counts points, time, and moves the character across the screen
- Feature 5 - Start and end
  - The start screen when the program is run, and the end screen that gives the player score and the option to play again
 
## Challenges
- How to create a user input mechanic and set it to click
  - Mechanic actives physics
- Set a start and an ending screen outside of the game loop
- Create art that moves with the respective mechanics where applicable

## Outcomes
Ideal Outcome:
  - To create a game that utilizes simple mechanics to make a character that jumps using a basic user input and ends the loop upon colliding with an obstacle, art and animations included

Minimum Viable Outcome:
  - To create a simple game that makes a square move up and down through obstacles while keeping track of score, and replayability
